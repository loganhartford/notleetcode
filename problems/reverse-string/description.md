# Reverse String

Write a function that reverses a string. The input is given as a **list of characters** `s`.

You must do this by modifying the input array **in place** with `O(1)` extra memory. Return `None` (do not return a new list).

### Example 1
```
Input:  s = ["h","e","l","l","o"]
After:  s == ["o","l","l","e","h"]
```

### Example 2
```
Input:  s = ["H","a","n","n","a","h"]
After:  s == ["h","a","n","n","a","H"]
```

### Constraints
- `1 <= len(s) <= 10^5`
- `s[i]` is a printable ASCII character.
